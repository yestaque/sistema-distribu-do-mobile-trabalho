📱 Sobre o Sistema

Tecnologias utilizadas:

✅ HTML5 + CSS3 + Bootstrap (responsivo)
✅ JavaScript puro (ES6+)
✅ React Native (app mobile nativo)
✅ SQLite via localStorage / AsyncStorage
✅ API RESTful (Open-Meteo, Nominatim)
✅ Banco de dados em nuvem (Firebase/Supabase)
✅ Sensor GPS do dispositivo (geolocalização)
✅ Pagamento PIX + Mercado Pago
✅ CRUD completo (usuários, produtos, pedidos)
